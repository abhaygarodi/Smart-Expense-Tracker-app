#  React Expense Tracker App

A user-friendly **Expense Tracker** built using **React**.  
It allows users to add new expenses, view them in a clean card layout, and delete entries when needed.  
The app features a modern UI with formatted dates, amount display, and a smooth data-handling flow using React hooks.

##  Features
- Add expenses with title, amount, and date  
- Delete any expense from the list  
- Beautiful card-style UI for each expense  
- Responsive and clean layout  
- State management using React's `useState`

##  Tech Stack
- React.js  
- JavaScript (ES6)  
- CSS (custom styling)

##  How to Run
```bash
npm install
npm run dev
```
